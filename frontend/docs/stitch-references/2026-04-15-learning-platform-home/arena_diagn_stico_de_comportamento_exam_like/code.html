CONCEPTUAL_EXPLANATION

O código demonstra Dynamic Method Dispatch. Embora a referência b seja do tipo Base, o objeto instanciado é Main.

TECHNICAL_CITATION

"An instance method m1 declared in a class C overrides another instance method m2 if C is a subclass of the class in which m2 is declared..." (JLS §8.4.8.1)

smart_toy AI_REVIEW

O compilador Java não exige que o método que sobrescreve declare as mesmas exceções que o método pai, desde que a nova declaração seja mais restritiva ou igual. Como Main.execute() remove a cláusula throws, o código é válido.
